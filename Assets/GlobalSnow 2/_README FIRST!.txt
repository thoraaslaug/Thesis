Welcome!
This asset contains 2 versions of Global Snow. Please use the appropriate package according to the rendering pipeline of your project:

** Builtin Pipeline version **
 Import package from /BuiltIn folder and add Global Snow script to your camera. That's all!
 IMPORTANT: always read documentation for detailed instructions.

** Universal Rendering Pipeline (URP) version **
 Import package from /URP folder and follow the instructions in the Documentation folder.
